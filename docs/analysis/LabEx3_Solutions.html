Question 2:

Write a program that computes the factorial of an integer. To find out more about factorial of a number, click here. The following steps can guide you to write your program:

(a) Prompt user for a number.

(b) If entry is invalid, end of program.

(c) If entry is a float (you can compare the type of entry) or entry is less than 0, end of program.

(d) Else compute the factorial as follows (assuming the number is n, n! denotes the factorial of n):

    n! = n x (n-1) x (n-2) x (n-3) x...1

Take note that the factorial of 0 is 1.

Question 3: (You many like to refer to Example 6E in Lab3 for reference)

Write a program that outputs all the possible three-character permuations (repetitions allowed) given four characters. For more on permutations with repetitions, click here. The following steps can guide you to write your program:

(a) Prompt user to enter four distinct characters one at a time.

(b) If the length of any characters is more than 1, end of program.

(c) Else if there is any matching pair of characters, end of program.

(d) Else concatenate all four characters. Declare an empty list and proceed to step (e).

(e) Iterate through the string using three nested loops such that you exhaust all possible permutations, appending each permutation to the list each time.

(f) Output the list and its length.

        grade = "A"
    elif (60 < score <= 80):
        grade = "B"
    elif (40 < score <= 60):
        grade = "C"
    elif (20 < score <= 40):
        grade = "D"
    elif (0 <= score <= 20):
        grade = "F"
    else: 
        grade = "not applicable"
        
    print("Your grade is " + grade + ".")
    
except:
    print("You did not enter a valid score. Good-bye!")

try:
    num = eval(input('Enter a positive integer: '))
    if (type(num) == float) or num < 0:
        print("You did not enter a valid number. Good-bye!")
    else:
        fac = 1
        for i in range(1,num+1):
            fac *= i           
        print("The factorial of {} is {}.".format(num, fac))
except:
    print("You did not enter a valid number. Good-bye!")
